# Strategy

## Current Focus: Landing Page Launch

Building the launch landing page for [AppName], the AI voice dictation app for Mac.

### Strategic Priorities

1. **Build a high-converting landing page** that communicates speed, simplicity, and the unique audio upload feature
2. **Position against Wispr Flow** with direct speed comparisons and better pricing
3. **Leverage existing distribution** (438K TikTok, creator/agency network) for organic launch
4. **Capture waitlist signups** for Windows/iOS to gauge demand

### Landing Page Goals

- Communicate core value prop: "Fastest AI voice dictation on Mac"
- Make audio file upload feature prominent (key differentiator)
- Drive Mac downloads and free tier signups
- Capture email/waitlist for Windows/iOS
- Establish premium, creator-first brand positioning

### Page Sections

1. Hero (speed + simplicity messaging, CTA)
2. How It Works (3-step flow)
3. Key Features (dictation, audio upload, universal support, tone matching)
4. Speed Comparison (vs Wispr Flow, Apple, Google)
5. Use Cases (agency owners, creators, developers, sales/support)
6. Pricing (Free, Pro, Creator, Teams)
7. Testimonials/Social Proof
8. FAQ
9. Final CTA
10. Footer

### Design Direction

- Dark mode primary (dark navy/charcoal)
- Vibrant accent (electric blue, teal, or warm amber)
- Apple-level polish, no generic SaaS look
- Distinctive typography (not Inter/Roboto)
- Sound wave/waveform as visual motif
- Micro-animations, generous whitespace
- Mobile responsive, fast loading

### Go-to-Market (Post-Landing Page)

**Phase 1: Launch**
- Organic TikTok content (speed comparison demos)
- Product Hunt launch
- Direct outreach to creator/agency communities

**Phase 2: Growth**
- Content marketing (TikTok, LinkedIn, YouTube Shorts)
- AI/productivity influencer partnerships
- SEO blog content
- Referral program

**Phase 3: Expansion**
- Windows and iOS release
- Team/Enterprise features
- API access, integrations

### Success Metrics

- Landing page conversion rate (visitor → download/signup)
- Free tier signups
- Waitlist signups (Windows/iOS)
- Email capture rate
- Time on page, scroll depth
