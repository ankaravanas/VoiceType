# Section Copy Patterns

Detailed writing patterns for each section type with real examples from WeDoHype projects. Use as reference when writing any specific section.

---

## Hero Section

The hero is the most important section. It determines whether visitors stay or bounce.

### Headline Structures

**1. Benefit-Promise (strongest for services):**
- "Reduce friction for you and your customers with smart Automations & Email Marketing."
- "Σταμάτα να εξαρτάσαι ΜΟΝΟ από online platforms. Αύξησε τις κρατήσεις σου με ένα website και 0% προμήθεια."

**2. Problem-Reframe (strongest for medical/health):**
- "Η ανάρρωση δεν είναι θέμα υπομονής. Είναι θέμα επιστημονικής παρέμβασης."
- "Δεν χρειάζεται να τα ξέρεις όλα. Αρκεί να έχεις τη σωστή υποστήριξη."

**3. Identity Statement (strongest for products/brands):**
- "Η επιστήμη της φυσικής φροντίδας του δέρματος."
- "Το τσίπουρο που κουβαλάει τη γεύση του τόπου και την τέχνη της παράδοσης."
- "This is how Athens does pizza bars."

**4. Question (strong for recognition):**
- "Γιατί κάθε επίσκεψη στο κομμωτήριο μοιάζει με ρίσκο;"

**5. Command (strong for urgency):**
- "Πες όχι σε ανεπαρκή απολύμανση και κινδύνους υγιεινής."

### Hero Subheading Rules
- Expand on the headline, don't repeat it
- Add the "how" or "what" to the headline's "why"
- 1-2 sentences max
- Include a specific detail: number, method, or proof point

### Hero CTA Rules
- Same CTA that will repeat throughout the page
- Action verb + specificity: "Κλείσε δωρεάν ραντεβού", not "Μάθε περισσότερα"
- Optional: secondary CTA link below ("Δες πώς λειτουργεί")

## Pain Points Section

The engine of conversion. If the visitor doesn't feel "that's me," they leave.

### Pain Point Framing Approaches

**Approach 1: Recognition Questions**
Best for lifestyle, beauty, parenting:
```
Heading: "Μήπως κάτι από αυτά σού είναι γνώριμο;"
- "Σου έχει τύχει να σου καταστρέψουν τα μαλλιά από λάθος τεχνική;"
- "Πόσες φορές είπες 'Δεν βγήκε όπως το φανταζόμουν';"
- "Νιώθεις ότι τα μαλλιά σου δεν σε αντιπροσωπεύουν;"
```

**Approach 2: "Say No To..." List**
Best for B2B, equipment, professional:
```
Heading: "Πες όχι σε..."
- Ανεπαρκή απολύμανση και κινδύνους υγιεινής
- Υψηλή κατανάλωση νερού, ενέργειας και απορρυπαντικών
- Συχνές βλάβες κι επισκευές
```

**Approach 3: Emotional Scenario**
Best for parenting, mental health, personal services:
```
Heading: "Τα σημάδια που αναγνωρίζεις..."
"Εκτοξεύσεις, άρνηση, δυσκολίες στον ύπνο, διαφωνίες...
Οι καθημερινές προκλήσεις σε κάνουν να αμφιβάλλεις.
'Δεν κάνω αρκετά;' 'Φταίω εγώ;'"
```

**Approach 4: Competitive Threat**
Best for B2B logistics, SaaS, agencies:
```
"Αν ένα άλλο κατάστημα εγγυάται ταχύτερη παράδοση, θα το επιλέξουν."
"Αν δεν τους παραδώσεις στην ώρα σου, δεν σε ξαναπροτιμούν."
```

**Approach 5: Problem/Solution Pairs**
Best for consultants, SaaS:
```
Problem: "I don't get enough sales from emails."
Solution: "Automation strategy + compelling copywriting = predictable conversions."
```

**Approach 6: Symptom List**
Best for medical:
```
"Μην αγνοείς τα σημάδια που επηρεάζουν την καθημερινότητά σου."
- Δυσκολία στην αναπνοή
- Πίεση ή πονοκέφαλοι στο πρόσωπο
- Ροχαλητό ή υπνική άπνοια
```

### Pain Points Rules
- Be specific to the industry, never generic
- Use the audience's actual vocabulary
- 3-5 pain points per section (not more)
- Each pain point should trigger recognition: "That's exactly my problem"

## Benefits / Features Section

Where you prove you can solve the problems from Section 2.

### Structure Pattern
Each benefit block:
```
**Subheading:** [Benefit name - short, declarative]
**Body:** [1-2 sentences with feature proof]
```

### Writing Rules
- Lead with the benefit, support with the feature
- "Απόλυτη υγιεινή" (benefit) → "Θερμοκρασίες έως 93°C" (feature proof)
- 4-6 benefits per section (not more - less is more)
- Each benefit should be independently valuable

### Benefit Subheading Examples
- "Αμεσότητα σε κάθε παραγγελία" (XP Blue - courier speed)
- "Εξοικονόμηση πόρων χωρίς συμβιβασμούς" (SMEG - cost savings)
- "Ομαλή μετάβαση χωρίς downtime" (Algomo - seamless integration)

## Process / How It Works Section

Reduces perceived complexity. Makes the next step feel easy.

### Format
3-4 steps (never more than 5):
```
Step 1: [Verb phrase] - [1-sentence explanation]
Step 2: [Verb phrase] - [1-sentence explanation]
Step 3: [Verb phrase] - [1-sentence explanation]
```

### Examples
**Consultant (Nontas):** Audit → Strategy → Implementation → Optimization
**Medical (PostAesthetics):** Assessment → Personalized Plan → Treatment → Recovery Support
**SaaS (Yummio):** Sign Up → Customize → Launch → Grow

### Rules
- Start each step with a verb
- Make each step feel effortless
- The overall message: "It's simpler than you think"

## Testimonials / Social Proof Section

### Testimonial Formats

**Format 1: Quote + Attribution (standard)**
```
"The Journey changed the way we interact as a team.
We opened up, we laughed, we had tough conversations."
 - HR Director, Financial Services
```

**Format 2: Result-led + Quote (strongest)**
```
**"3x conversions in 2 months!"** - Fred K.
"Algomo completely transformed our website's performance."
```

**Format 3: Multi-platform ratings (medical)**
```
Doctoranytime 10/10 | Google 5/5 | Instadoctor 5/5
95% saw improvement from the first visit.
```

### Statistics Bar Examples
- "800M views σε social channels" | "100M+ EUR client revenue" | "100+ projects"
- "4.000+ οικογένειες" | "300+ πελάτες" | "3.100+ 5-star κριτικές"
- "69 Client Companies" | "117 Programs" | "4,377 Leaders"

### Social Proof Rules
- Named testimonials > anonymous
- Specific results > vague praise ("Game-Changer!" is banned)
- Numbers with context > numbers alone
- Multi-platform ratings boost credibility significantly
- Include a CTA after testimonials (the reader is at peak trust)

## FAQ Section

### Question Type Mix
- **40% Practical:** "Πόσο κοστίζει;" / "Πόσο διαρκεί;" / "Κάνετε delivery;"
- **35% Objection-handling:** "Γιατί να έρθω σε εσάς;" / "Is this just team building?"
- **25% Educational:** "Τι είναι η τεχνολογία CEREC;" / "What is MLD?"

### Answer Style
- Short (1-2 sentences) for practical questions
- Medium (3-4 sentences) for objections - explanation + reassurance
- Detailed (paragraph) for educational - but still concise

### FAQ Count by Page Type
- Landing Page: 6-10 questions
- Homepage: 6-8 questions
- Ecommerce: 8-12 questions (heavier on shipping, returns, ingredients)

### FAQ Close
Always end with: "Δεν βρήκες την απάντηση; Επικοινώνησε μαζί μας!" or "Still have questions? Reach out!"

## CTA Section

### Standalone CTA Section (mid-page)
- **Heading:** Action-oriented or emotional appeal
- **Body:** 1 sentence reducing friction ("Χωρίς δέσμευση" / "Free consultation")
- **CTA Button:** Primary action
- **Form:** Name + Email + Phone + Message (max 4-5 fields)

### Final CTA Section
- **Heading:** The most emotionally charged headline of the entire page
- **Body:** 1-2 sentences - urgency, reassurance, or emotional pull
- **CTA Button:** Same as Hero (consistency)

### CTA Headline Examples (Final sections)
- "Όχι άλλη προσωρινή ανακούφιση. Χάρισε στο σώμα σου την ανάρρωση που του αξίζει."
- "Πόσο πιο εύκολα θα πατούσαν 'Ολοκλήρωση Παραγγελίας' οι πελάτες σου;"
- "Ας γνωριστούμε!"

### CTA Repetition Pattern
Repeat the primary CTA button 3-5 times through the page:
1. After Hero
2. After Pain Points or Solution
3. Standalone CTA + Form section (mid-page)
4. After Testimonials
5. Final CTA section

## Newsletter / Lead Capture Section

### Format
- **Heading:** Offer-driven, not "Subscribe to our newsletter"
- **Body:** 1 sentence - what they get
- **CTA:** Action-specific
- **Form:** Email only (maximum: email + first name)

### Examples
- Heading: "10% έκπτωση στην πρώτη σου παραγγελία"
- CTA: "Θέλω 10% έκπτωση!" or "Εγγραφή"

## Comparison Table Section

Shows why the brand is the better choice. Must be a native Google Docs table, not markdown.

### Table Structure
Always 3 columns:
```
Column 1: Feature/aspect (short phrase, one line)
Column 2: [Brand Name] → ✓ or short phrase
Column 3: Οι υπόλοιποι / Μόνος σου → ✗ or short phrase
```

### Preferred Format: Short Phrases + ✓/✗
The best comparison tables are **visually scannable**. Each row is a short phrase (max one line), and columns 2 and 3 use ✓ / ✗ marks. This is visually cleaner and more effective than long descriptions.

### Heading Pattern
Frame as a question or direct comparison:
- "[Brand] ή μόνος/η σου;"
- "Γιατί [Brand] και όχι κάτι άλλο;"
- "Δες τη διαφορά."

### Row Content Rules
- 5-8 rows (not more)
- Each row is ONE short phrase. One line max. No sentences, no explanations.
- Keep it professional but human. Not corporate, not χύμα.
- NEVER assume features the brand hasn't confirmed (e.g. don't write "Dashboard σε real-time" unless you know they have one)
- Be honest about competitors. Some rows can be ✓/✓ if competitors also offer it.

### Writing Rules
- NEVER use English business jargon in Greek copy (no "cash flow management", "financial clarity", etc.)
- Write as if explaining to someone with zero industry knowledge
- Each feature must be something the reader can picture in their daily life
- Focus on differences that actually matter to the buyer

### Examples

**Service Business (Quantro, Fractional CFO):**
```
| Θέμα | Quantro | Μόνος σου |
| Πλήρης εικόνα κερδοφορίας | ✓ | ✗ |
| Forecasting ρευστότητας | ✓ | ✗ |
| Προϋπολογισμός βασισμένος σε data | ✓ | ✗ |
| Αξιολόγηση επενδύσεων και προσλήψεων | ✓ | ✗ |
| Fundraising preparation | ✓ | ✗ |
| Χωρίς κόστος full-time πρόσληψης | ✓ | ✗ |
```

**Tourism (Milos Rent Boat):**
```
| Feature | Milos Rent Boat | Others |
| No-License Boat Rentals | ✓ | ✗ |
| Private, Small-Group Focus | ✓ | ✗ |
| Newest Boats (2023 Models) | ✓ | ✗ |
| Customizable Private Tours | ✓ | ✗ |
| Transparent Pricing | ✓ | Some Add-on Charges |
| Local Skippers with Secret Spots | ✓ | Tourist Routes Only |
```

**B2B SaaS (CarbonBox SOP):**
```
| Points | Facebook Media Buying SOP | Others |
| Data-Driven Advertising | ✓ | Maybe |
| Continuous Optimization | ✓ | ✗ |
| A/B Testing | ✓ | ✗ |
| Performance Analysis | ✓ | Maybe |
```

## About / Origin Story Section

### Storytelling Pattern
1. **The trigger moment:** What specific event or realization started everything
2. **The challenge:** What obstacles were overcome
3. **The mission:** What drives the company today
4. **The proof:** Where they are now (numbers, reach, impact)

### Examples
- Auraskin: Coffee waste discovery at University of Patras → 2 years R&D → first Greek recycled-coffee skincare line
- Meetato: Greek "mitato" (shepherd hut) + "meet" → corporate retreat brand connecting nature and teamwork

### Rules
- Use real names, places, dates
- Be specific: "στο Πανεπιστήμιο Πατρών" not "σε ένα πανεπιστήμιο"
- Personal voice: written as if the founder is telling the story
- 3-5 paragraphs for About pages, 2-3 for Homepage teasers
